Say goodbye to unpleasant words in you blog comments!
== Description ==

With this simple plugin you can choose unpleasant words and change them them to some words you like automaticaly.!

*	Filter all of the unpleasant words in past or future
*	Change all of the unpleasant words to a word that you like

== Installation ==

1. Extract the downloaded archive into `comment_filter` folder. If you downloaded the .zip archive it **contains** the `comment_filter` folder.
2. Upload `comment_filter` to the `/wp-content/plugins/` directory of your wordpress installation.
3. If you are using older versions of this plug-in, you must deactivate it first through the 'Plugins' menu in Wordpress admin.
4. Activate the plug-in through the 'Plugins' menu in Wordpress admin.
